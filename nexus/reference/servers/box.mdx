## Overview

The Box server enables AI assistants to interact with [Box](https://www.box.com) cloud storage, providing seamless file management, document retrieval, and content analysis capabilities through OAuth-secured connections.

## How to Add Box

<Steps>
  <Step title="Create Box Account">
    1. Go to [https://account.box.com/](https://account.box.com/)
    2. Sign up for a free Box account or use your existing account
    3. Upload files to your Box storage
  </Step>
  <Step title="Connect to Nexus">
    1. Add the Box server to your Nexus environment through the server directory
    2. Authorize Box when prompted using OAuth
  </Step>
  <Step title="Test Connection">
    Start with a simple command like "Show me my box files" to verify the connection works properly.
  </Step>
</Steps>

<Tip>
Box uses remotely-hosted MCP server with OAuth authorization. For detailed setup instructions, visit [Managing Box MCP Servers](https://support.box.com/hc/en-us/articles/43847256139923-Managing-Box-MCP-Servers).
</Tip>

## Use Cases

### Document Discovery
"Show me my box files" - Quickly browse all files stored in your Box account.

### Data Extraction
"Extract all the names and email addresses from this document" - Use AI to parse structured data from documents.

### Content Search
"Search for files containing 'contract'" - Find relevant documents across your entire Box storage.

### File Type Filtering
"Find all Word documents in my Box account" - Locate files by specific formats.

<Note>
Box supports OAuth authorization and provides secure access to your cloud storage. The server is remotely hosted by Box for optimal performance and security.
</Note>
