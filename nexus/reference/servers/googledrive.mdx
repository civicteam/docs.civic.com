## Overview

The Google Drive server provides comprehensive integration with [Google Drive](https://drive.google.com), enabling AI assistants to search, access, create, and manage files and folders across personal drives and shared drives.

## Getting Started

<Steps>
  <Step title="Connect to Google Drive">
    Add the Google Drive server to your Nexus environment through the server directory.
  </Step>
  <Step title="Test Connection">
    Start with a simple command like `search for files` or `list files in root folder` to verify the connection works properly.
  </Step>
</Steps>

## Advanced Features

### Smart Content Extraction
- **Google Workspace Files**: Native export of Docs, Sheets, and Slides to readable text/CSV
- **Office Documents**: Automatic parsing of .docx, .xlsx, .pptx files with text extraction
- **Universal Support**: UTF-8 text extraction with binary file detection

### Shared Drive Support
- **Multi-Drive Search**: Search across personal drives and accessible shared drives
- **Drive-Specific Operations**: Target specific shared drives for focused operations
- **Permission-Aware**: Respects user permissions and access levels

### Search Capabilities
- **Google Search Operators**: Full support for Drive's native search syntax
- **Metadata Filtering**: Search by file type, owner, modification date, and more
- **Pagination Support**: Handle large result sets with efficient pagination

<Note>
The Google Drive server requires appropriate Google Workspace permissions and OAuth2 authentication to access and modify files in your Google Drive.
</Note>
