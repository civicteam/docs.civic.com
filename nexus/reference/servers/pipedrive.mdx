## Overview

The Pipedrive server enables AI assistants to interact with [Pipedrive](https://www.pipedrive.com) CRM, providing complete sales pipeline management, contact organization, and deal tracking capabilities through conversational AI.

## How to Add Pipedrive

<Steps>
  <Step title="Create Pipedrive Account">
    1. Go to [https://www.pipedrive.com](https://www.pipedrive.com)
    2. Sign up for an account or log in to your existing workspace
  </Step>
  <Step title="Generate API Key">
    1. Click your avatar and go to **Personal preferences**
    2. Open the **API** tab
    3. Generate a new API token
    4. Copy the token for use in Nexus
  </Step>
  <Step title="Connect to Nexus">
    1. Add the Pipedrive server to your Nexus environment
    2. Enter your API token when prompted
  </Step>
  <Step title="Test Connection">
    Start with a simple command like "List my contacts on Pipedrive" to verify the connection works properly.
  </Step>
</Steps>

<Tip>
OAuth support will be added once Civic's marketplace app is approved by Pipedrive. Currently, API key authentication is available.
</Tip>

## Use Cases

### Create Organization and Contact
"Create an organization called 'ABC Systems' on Pipedrive. Dave Jones is the first contact person (dave.jones@example.com)" - Set up new prospects with complete information.

### List Contacts
"List my contacts on Pipedrive" - Review all contacts in your CRM.

### Create Deal
"Create a deal to supply workflow automation agents to ABC Systems to the value of $50k, with Dave Jones as the contact" - Track new opportunities with full context.

### Move Deal Through Pipeline
"Move the ABC Systems automation deal into the 'Demo scheduled' stage" - Update deal status as it progresses.

<Note>
Pipedrive uses a Civic fork of the [bratland/pipedrive-mcp-server](https://github.com/bratland/pipedrive-mcp-server) repository with added write operation tools for comprehensive CRM management.
</Note>
