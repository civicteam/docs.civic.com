## Overview

Apify is a web scraping and automation platform that provides pre-built Actors (scrapers, crawlers, and AI agents) for extracting data from websites. Connect Civic to Apify to enable AI-driven browsing, actor execution, and documentation search.

<Card title="Add to Civic" icon="plus" href="https://app.civic.com">
  Connect Apify at app.civic.com
</Card>

## Available Tools (7)

| Tool | Description |
|------|-------------|
| `apify-slash-rag-web-browser` | Web browser for RAG pipelines and AI agents — queries Google Search, scrapes top N pages, and returns content as Markdown |
| `call-actor` | Call any Actor from the Apify Store using a two-step workflow: first get the Actor's schema, then run it with proper input |
| `fetch-actor-details` | Get detailed information about an Actor by ID or full name, including README, input schema, pricing, and stats |
| `fetch-apify-docs` | Fetch the full content of an Apify documentation page by its URL |
| `get-actor-output` | Retrieve the output dataset items of a specific Actor run using its datasetId, with field selection and pagination |
| `search-actors` | Search the Apify Store to find pre-built Actors for specific platforms or use cases |
| `search-apify-docs` | Search Apify documentation using full-text search to find relevant pages by keywords |
