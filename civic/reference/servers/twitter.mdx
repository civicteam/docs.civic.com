## Overview

The Twitter server enables AI assistants to interact with the X (formerly Twitter) API to post new tweets with optional media attachments, delete existing tweets, and search public content.

## Available Tools (3)

| Tool | Description |
|------|-------------|
| `post_tweet` | Post a new tweet to X with optional media attachments (up to 4 images) |
| `delete_tweet` | Delete a tweet by its ID |
| `search_tweets` | Search for tweets on X using a query string |
