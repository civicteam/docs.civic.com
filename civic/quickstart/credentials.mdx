## Your MCP URL

Every Civic connection uses the same base URL:

```
https://nexus.civic.com/hub/mcp
```

### URL Parameters

Append these to the base URL to control agent behavior:

| Parameter | Description | Example |
|-----------|-------------|---------|
| `profile` | Lock the agent to a specific toolkit | `?profile=support-agent` |
| `lock` | Block toolkit switching (default: `true` when `profile` is set) | `?profile=support&lock=true` |
| `skills` | Pre-load one or more skills | `?skills=escalation,canned-responses` |
| `accountId` | For multi-account users or organizations | `?accountId=org_abc123` |

**Example with profile locking:**
```
https://nexus.civic.com/hub/mcp?profile=support-agent&lock=true
```

## Authentication Methods

Civic supports two credential types depending on your use case:

| Client Type | Method | When to Use |
|-------------|--------|-------------|
| Interactive clients (Claude Desktop, Cursor, etc.) | **OAuth** | Human-in-the-loop workflows |
| Agents / automation / API calls | **Bearer Token** | Programmatic / unattended access |

### OAuth (Interactive Clients)

OAuth is handled automatically — most clients prompt you to authenticate in a browser on first connection. No manual credential setup required.

### Bearer Token (Agents & Automation)

Generate a token at [nexus.civic.com](https://nexus.civic.com):

1. Log in and go to **Settings → Tokens**
2. Click **Generate Token**
3. Copy the token — it's shown once

Use the token as a Bearer header in your agent:

```
Authorization: Bearer <your-token>
```

<Note>
Tokens expire after **30 days**. Regenerate before expiry to avoid disruptions — the old token becomes invalid immediately.
</Note>

Set it as an environment variable to avoid hardcoding:

```bash
CIVIC_TOKEN=your_token_here
CIVIC_URL=https://nexus.civic.com/hub/mcp?profile=your-toolkit
```

## For New App Builders

If you're building an app where **your users** need to connect their own accounts to Civic, use **Civic Auth** to identify them — then exchange their token for a Civic access token.
