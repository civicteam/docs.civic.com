### 🛡️ Enterprise Security & SDK Enhancements

Major security improvements for enterprise applications and expanded framework support.

- **Client secret authentication** - Full support for confidential client applications with enhanced security
- **Universal JavaScript SDK** - Enhanced vanilla JavaScript support compatible with any backend framework

- **Extended refresh tokens** - Token expiration extended to 400 days for seamless long-term sessions
- **Optimized mobile login** - Improved provider display and faster loading on mobile devices

- **Authentication redirect fixes** - Resolved login app loading issues during authentication flows
- **Account management fixes** - Eliminated duplicate account creation in Customer Dashboard
- **Mobile display enhancements** - Fixed logo pixelation and provider icon display issues
- **Apple authentication** - Better error handling when users cancel Apple login flow
