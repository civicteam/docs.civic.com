## What are Recipes?

Recipes are practical, step-by-step guides that show you how to implement specific features or integration patterns with Civic Auth. Each recipe includes:

- Complete working code examples
- Clear explanations of each implementation step
- Common pitfalls and how to avoid them
- Best practices for production deployments
