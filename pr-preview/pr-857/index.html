Civic Documentation

Civic is the agent integrator for mid-market businesses. We ship forward deployed agents — production agents scoped to your workflow, built on your stack, governed from day one.

These docs cover the Civic platform: the infrastructure, tools, and open-source components that power every agent we deploy.

Civic Hub

MCP gateway with guardrails, audit logs, secret management, and instant revocation. Connect agents to your tools securely.

Civic Auth

Authentication for web and mobile apps. Email, Google, and Web3 wallet sign-in with embedded wallets on Solana and EVM.

Civic Labs

Open-source agent infrastructure. MCP Hub, Guardrail Proxy, Bodyguard, and more — the building blocks behind forward deployed agents.

The Civic Method​

Every engagement follows the same disciplined process — whether you're a marketing agency, a law firm, or a manufacturer.

  1. 1
    Discovery

    45-minute call, free. Pick one workflow your team keeps postponing. You walk out with a one-page scope memo.

  2. 2
    Scope & Build

    On your stack. Built on your data, your SSO, your infrastructure. Civic components only where you want them.

  3. 3
    Shadow

    Human-in-the-loop validation. Your team reviews every agent action before it runs autonomously. Configurable checkpoints.

  4. 4
    Cutover

    A live production agent, documentation to manage it, and a full handover if you want to run it yourself.

Governance by Default​

Every agent Civic deploys ships inside a governance envelope. This is not an add-on — it's how we build.

Audit Trail

Complete log of every tool call — tool name, parameters, response, timestamp, agent identity. Queryable via Civic Chat.

Guardrails

Block tools, enforce read-only access, preset parameters. 14 built-in guardrails at the tool, server, and toolkit level.

Revocation & Kill Switch

Instant revocation at any granularity — a single tool, a server, or an entire toolkit. One click.

Secret Management

OAuth tokens, API keys, and bearer tokens stored in Civic. The agent never sees raw credentials.
