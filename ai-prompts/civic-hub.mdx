Use this prompt with your preferred AI assistant (Claude, Cursor, ChatGPT, etc.) to connect your AI agent to Civic Hub — the MCP gateway that provides guardrails, audit logging, secret management, and access to 85 MCP servers.

<Note>
**Civic Hub vs Civic Auth:** This prompt is for **agent security** (connecting AI agents to tools safely). If you need **user authentication** for a web app, see the [Civic Auth prompts](/ai-prompts/overview#available-framework-prompts) instead.
</Note>

## How to Use

1. **Copy the prompt below** and paste it into your AI assistant
2. **Tell the assistant** which agent framework you're using (Anthropic SDK, LangChain, OpenAI Agents, etc.)
3. **The assistant will walk you through** setting up your MCP connection with the correct code for your framework
4. **Configure security** at [app.civic.com](https://app.civic.com) — guardrails, toolkits, and audit are managed in the platform, not in code

</Warning>

## What the AI Assistant Will Do

When you use this prompt, the AI assistant will:

1. **Ask which framework** you're using (Anthropic SDK, OpenAI Agents, LangChain, etc.)
2. **Generate the connection code** — typically 5-15 lines to connect to Civic Hub via MCP
3. **Set up environment variables** for your MCP URL and token
4. **Point you to app.civic.com** for guardrail and toolkit configuration
