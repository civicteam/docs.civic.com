## Overview

PayPal is a global payments platform for managing orders, invoices, subscriptions, products, disputes, and refunds. Connect it to Civic to let your AI agent handle the full PayPal payment lifecycle including billing plans and shipment tracking.

<Card title="Add to Civic" icon="plus" href="https://app.civic.com">
  Connect PayPal at app.civic.com
</Card>

## Available Tools (28)

| Tool | Description |
|------|-------------|
| `accept_dispute_claim` | Accept liability for a dispute claim, closing it in the customer's favor |
| `cancel_sent_invoice` | Cancel a sent invoice |
| `cancel_subscription` | Cancel a customer subscription |

| `get_invoice` | Get an invoice from PayPal |
