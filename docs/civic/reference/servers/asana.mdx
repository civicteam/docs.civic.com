## Overview

Asana is a project and task management platform used by teams to plan, track, and manage work. Connect Civic to Asana to create and update tasks, manage projects and goals, organize portfolios, and coordinate team workflows.

<Card title="Add to Civic" icon="plus" href="https://app.civic.com">
  Connect Asana at app.civic.com
</Card>

## Available Tools (43)

| Tool | Description |
|------|-------------|
| `asana_add_task_followers` | Add followers to a task to notify them of updates |
| `asana_create_goal` | Create a new goal with a name, time period, and workspace or team |
| `asana_create_project` | Create a new project in a workspace or team |
| `asana_create_project_status` | Create a project status update with a color and text |
| `asana_create_task` | Create a task in a project, as a subtask, or in a workspace |
| `asana_create_task_story` | Add an explicit comment to a task |
| `asana_delete_task` | Delete a task from Asana |
| `asana_get_attachment` | Get detailed data for a specific attachment |
| `asana_get_attachments_for_object` | List all attachments for a project, project brief, or task |
| `asana_get_goal` | Get detailed goal data including name, owner, and current status |
| `asana_get_goals` | List goals filtered by portfolio, project, task, workspace, or team |
| `asana_get_items_for_portfolio` | List projects, goals, and other items in a portfolio |
| `asana_get_parent_goals_for_goal` | List all parent goals for a specific goal to understand goal hierarchy |
| `asana_get_portfolio` | Get detailed portfolio data by ID including name, owner, and projects |
| `asana_get_portfolios` | List portfolios filtered by workspace and owner |
| `asana_get_project` | Get detailed project data including name, description, owner, members, and custom fields |
| `asana_get_project_sections` | List all sections in a project |
| `asana_get_project_status` | Get a single project status update by ID |
| `asana_get_project_statuses` | List all status updates for a project chronologically |
| `asana_get_project_task_counts` | Get task statistics for a project — incomplete, completed, and milestone counts |
| `asana_get_projects` | List projects filtered by workspace |
| `asana_get_projects_for_team` | List all projects for a team |
| `asana_get_projects_for_workspace` | Get all projects in a workspace across all teams |
| `asana_get_stories_for_task` | Get task activity history including comments, status changes, and system events |
| `asana_get_task` | Get full task details by ID including name, assignee, due dates, and dependencies |
| `asana_get_tasks` | List tasks filtered by workspace, project, section, tag, or user list |
| `asana_get_team_users` | List all members of a team |
| `asana_get_teams_for_user` | Get teams a user belongs to within an organization |
| `asana_get_teams_for_workspace` | List all teams in a workspace |
| `asana_get_time_period` | Get detailed information about a time period by ID |
| `asana_get_time_periods` | List available time periods in a workspace |
| `asana_get_user` | Get user details by ID, email, or `"me"` |
| `asana_get_workspace_users` | Get all users in a workspace |
| `asana_list_workspaces` | Get all accessible workspaces — call this first to discover valid workspace IDs |
| `asana_remove_task_followers` | Remove followers from a task to stop notification subscriptions |
| `asana_search_tasks` | Advanced task search with filters for text, dates, status, users, projects, portfolios, and custom fields |
| `asana_set_parent_for_task` | Change a task's parent to convert it to or from a subtask |
| `asana_set_task_dependencies` | Set tasks that this task depends on (prerequisites) |
| `asana_set_task_dependents` | Set tasks that are blocked by this task |
| `asana_typeahead_search` | Quick search across Asana objects by type — use this first before specialized search tools |
| `asana_update_goal` | Update goal properties including name, owner, dates, metrics, or status |
| `asana_update_task` | Update existing task properties including assignee, due date, completion status, and name |
