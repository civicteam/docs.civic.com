## Overview

The Tavily server provides AI-optimized web search and research tools. Search for current information, crawl websites for content, extract raw page text, map site structure, and run comprehensive multi-source research tasks.

<Card title="Add to Civic" icon="plus" href="https://app.civic.com">
  Connect Tavily at app.civic.com
</Card>

## Available Tools (5)

| Tool | Description |
|------|-------------|
| `tavily_search` | Search the web for current information and return snippets with source URLs |
| `tavily_crawl` | Crawl a website starting from a URL, extracting content with configurable depth and breadth |
| `tavily_extract` | Extract raw content from a list of URLs in markdown or text format |
| `tavily_map` | Map a website's structure and return a list of URLs found from the base URL |
| `tavily_research` | Perform comprehensive multi-source research on a topic and return a detailed response |
