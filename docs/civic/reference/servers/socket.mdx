## Overview

The Socket server enables AI assistants to evaluate the security and quality of software packages using Socket's dependency analysis platform, helping identify risky or compromised dependencies before they reach production.

## Available Tools (1)

| Tool | Description |
|------|-------------|
| `depscore` | Get the dependency score of packages for quality and security analysis across ecosystems (npm, pypi, etc.) |
