If you're ready to integrate our tools:

#### BodyGuard
```bash
# Use our hosted API (contact for API key)
curl -X POST https://api.civic.com/labs/bodyguard/check \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Analyze this prompt"}'
```

#### MCP Tools
```bash
# Coming soon: NPM packages for direct integration
npm install @civic/mcp-client
npm install @civic/passthrough-proxy
npm install @civic/guardrail-hook
```

## What You'll Need

Depending on which tools you want to use:

- **For BodyGuard**: OpenAI API key or access to our hosted service
- **For MCP Tools**: Docker environment or Node.js 18+
- **For Integration**: Basic familiarity with APIs and async programming
