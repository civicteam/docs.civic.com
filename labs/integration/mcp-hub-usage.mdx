## Usage

The MCP Hub can be integrated in several ways:

1. **Direct API Integration**: Connect your application directly to the Hub API
2. **Desktop Bridge**: Use the hub-bridge package to connect desktop LLM apps
3. **CLI Testing**: Use the mcp-client CLI tool for development and testing
4. **Container Sidecar**: Deploy MCP servers with the stdio transport adapter
5. **Authorization Manager**: Use the MCP server for managing auth tokens

## API Endpoint

The MCP Hub endpoint is:

```
https://ai.civic.com/hub/mcp
```

## Authentication

The MCP Hub supports multiple authentication methods:

### Bearer Token
Use your access token from Civic Auth:

```javascript
headers: {
  'Authorization': `Bearer ${accessToken}`
}
```

### API Key
Use your Civic Labs API key:

```javascript
headers: {
  'Authorization': `Bearer ${process.env.MCP_API_KEY}`
}
```

## MCP Protocol

The MCP Hub is compliant with the [MCP Protocol](https://modelcontextprotocol.io) specification.

[Contact us](/labs/feedback) for detailed API documentation and access.
