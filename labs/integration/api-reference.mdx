## Endpoint

The MCP Hub endpoint is:

```
https://ai.civic.com/hub/mcp
```

## Authentication

The MCP Hub supports multiple authentication methods:

### Bearer Token
Use your access token from Civic Auth:

```javascript
headers: {
  'Authorization': `Bearer ${accessToken}`
}
```

### API Key
Use your Civic Labs API key:

```javascript
headers: {
  'Authorization': `Bearer ${process.env.MCP_API_KEY}`
}
```

## Integration Examples

### Direct API Integration
Connect your application directly to the Hub API.

### Desktop Bridge
Use the hub-bridge package to connect desktop LLM apps.

### CLI Testing
Use the mcp-client CLI tool for development and testing.

### Container Sidecar
Deploy MCP servers with the stdio transport adapter.

### Authorization Manager
Use the MCP server for managing auth tokens.

## MCP Protocol

The MCP Hub is compliant with the [MCP Protocol](https://modelcontextprotocol.io) specification.
