## Overview

Civic Labs supports multiple authentication methods to suit different integration scenarios.

## Authentication Methods

### Civic Auth
For user-facing applications requiring delegated access:
- OAuth2 flow for secure user authentication
- Single sign-on across all tools
- Token lifecycle management

See the [Civic Auth documentation](https://docs.civic.com/auth) for detailed setup instructions.

### API Keys
For server-to-server communication:
- Simple bearer token authentication
- Suitable for backend services and automation

[Contact us](/labs/feedback) to obtain an API key.

## Setting Up Authentication

### With Civic Auth
```javascript
import { getTokens } from "@civic/auth/nextjs";

const { accessToken } = await getTokens();

// Use the token in your requests
headers: {
  'Authorization': `Bearer ${accessToken}`
}
```

### With API Keys
```javascript
// Store your API key securely
const apiKey = process.env.MCP_API_KEY;

// Use in requests
headers: {
  'Authorization': `Bearer ${apiKey}`
}
```
